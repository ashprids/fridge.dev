## Libraries

[rubberband-wasm v3.3.0](https://www.npmjs.com/package/rubberband-wasm) by Dani Biro, used for Rubber Band powered pitch and time processing.

The bundled Rubber Band WebAssembly license text is included at `/tools/frdgbeats/vendor/rubberband-wasm-LICENSE.txt`.

Everything else in frdgBeats is built with vanilla JavaScript and APIs that come with the browser.
